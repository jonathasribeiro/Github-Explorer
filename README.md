# GitHub Explorer

Single-page application for exploring GitHub repositories, built with **React** and **TypeScript**.

## Features

- Search and browse public GitHub repositories
- Repository detail view with metadata
- Responsive layout with styled-components
- GitHub REST API integration via Axios

## Stack

| Layer | Technology |
|-------|------------|
| UI | React |
| Language | TypeScript |
| Routing | React Router |
| HTTP | Axios |
| Styling | styled-components |

## Getting started

```bash
npm install
npm start
```

Open `http://localhost:3000` in your browser.
